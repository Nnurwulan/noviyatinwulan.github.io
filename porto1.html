Lecturer Information System

Tools

Laravel, PHP, CSS

The Brief

The development of information and management systems in tertiary institutions is very important. This is due to the increasing need for and the completeness of the attributes of each component related to academic affairs in tertiary institutions. Therefore, data management and good governance are needed so that information becomes accurate, transparent, accountable and up to date. This information system can certainly provide an overview and formulation to leaders in implementing policies towards lecturers.

because the sttnf campus previously had information in different places, so I want to unify all the information in one place.

The Result

It is hoped that the design of the lecturer information system website will make it easier for lecturers and academic administration to obtain data containing standard operating procedures.

Besides that, it can help lecturers to find the required information validly, quickly, and easily.

Lecturer information system includes several stages, namely finding information needed by lecturers and analyzing information needs. In addition, the implementation of a web-based information system uses PHP and the Laravel framework to make it easier to work on making the web so that the system will be more efficient and neat.
